\subsection*{Registers}
\textit{Each type (count, dimen, skip, muskip, toks) $\ni$ 255 registers. Access these directly, or, preferably, define an alias for what you'll use. Groups (\{\}) create a lexical stack of new registers.}\\
\entry{35mm}{\textbackslash count5=123}{set register}\\
\entry{35mm}{\textbackslash advance\textbackslash count by 1}{mutate register}\\
\entry{35mm}{\textbackslash multiply\textbackslash dimen4 by 3}{another mutate}\\
\entry{35mm}{\textbackslash newcount\textbackslash myname}{\say{reserve} a reg}\\
